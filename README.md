# AWS Security Learning Resources

Curated collection of AWS security workshops, hands-on labs, and GitHub repositories aligned with the Enterprise Cloud Security Knowledge Framework.

## Contents

- **AWS_Security_Learning_Resources.md** - Complete resource guide organized by security domains
- **AWS_Security_Resources_Table.csv** - Tabular format of all resources
- **AWS_Security_Resources_Table.xlsx** - Excel version with formatted columns
- **format_to_table.py** - Python script to generate tables from markdown

## Resources Coverage

104 total resources across 6 security domains:
- Identity and Access Management (IAM)
- Detection
- Infrastructure Protection
- Data Protection
- Incident Response
- Application Security
